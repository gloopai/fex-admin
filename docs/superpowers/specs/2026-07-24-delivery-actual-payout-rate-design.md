# 交割合约周期实际收益率设计

## 目标

在周期模板的每个周期中保留展示收益率 `payoutPct`，新增实际收益率 `actualPayoutPct`。界面展示两者，演示订单的盈利金额使用实际收益率计算；亏损订单仍损失全部本金。

## 数据与计算

- 周期结构：`{ id, durationSec, payoutPct, actualPayoutPct }`。
- `payoutPct` 仅用于产品档位展示。
- `actualPayoutPct` 用于演示收益计算：`毛收益 = 本金 × actualPayoutPct / 100`。
- 手续费仅按毛收益计算；猜错时返还为 0。
- 演示订单保存 `expectedYield`（展示收益率）和 `actualPayoutPct`（实际收益率）两个快照字段。

## 界面范围

- 周期模板新增、编辑、列表卡片和实时预览展示实际收益率。
- 合约管理的周期预览同时展示展示收益率和实际收益率。
- 订单 Mock 使用实际收益率生成预期收益和实际盈利。

## 验证

- 自动测试确认所有周期均包含有效的 `actualPayoutPct`。
- 自动测试确认订单收益计算引用实际收益率而非展示收益率。
- 运行完整测试与生产构建。
