## ⚡ 快速操作

### 场景 1: 创建新交易对（使用模板）

**步骤**:
```
1. 进入"汇率与费率"页面
   ↓
2. 确保在"汇率管理"Tab
   ↓
3. 点击右上角"+ 新增交易对"
   ↓
4. 在弹窗中：
   - 展开"基本信息"
   - 选择"💰 快速应用费率模板"
   - 选择"标准费率模板"
   ↓
5. 填写交易对信息:
   - 基础资产：USDT
   - 目标资产：BTC
   - 数据源：Binance
   - 市场汇率：0.000023
   ↓
6. 系统自动填充费率:
   - 买入费率：0.50%
   - 卖出费率：0.50%
   - 5 级分级费率
   ↓
7. 点击"保存" → MFA 验证 → 完成
```

**时间**: < 2 分钟

---

### 场景 2: 创建新的费率模板

**步骤**:
```
1. 切换到"费率模板"Tab
   ↓
2. 点击右上角"+ 新增模板"
   ↓
3. 在弹窗中：
   - 展开"基本信息"
     * 模板名称：我的优惠模板
     * 模板类型：优惠费率
     * 描述：适用于高流动性交易对
   ↓
4. 展开"基础手续费与价差"
     * 买入费率：0.30%
     * 卖出费率：0.30%
     * (自动同步到各级别)
   ↓
5. 展开"分级费率设置" (可选调整)
     * 微调各个等级的费率
   ↓
6. 点击"保存" → MFA 验证 → 完成
```

**时间**: < 3 分钟

---

### 场景 3: 为现有交易对更换模板

**步骤**:
```

   ↓

   ↓

6. 系统自动更新所有费率

```

**时间**: < 1 分钟

### 模板应用规则

**重要特性**:
- ✅ 应用模板是"复制"而非"链接"
- ✅ 修改模板不会自动更新已应用的交易对
- ✅ 交易对可以手动调整费率
- ✅ 调整后保持独立性

**使用建议**:
```
1. 先创建模板
2. 应用到多个交易对
3. 如需批量调整:
   - 方案 A: 逐个重新应用模板
   - 方案 B: 手动批量编辑
```

---

### 费率计算

**公式**:
```
平台买入价 = 市场汇率 × (1 + 买入费率)
平台卖出价 = 市场汇率 × (1 - 卖出费率)

示例:
市场价：1 BTC = 43,500 USDT
买入费率：0.5%
卖出费率：0.5%

→ 平台买入价：43,500 × (1 + 0.005) = 43,935 USDT
→ 平台卖出价：43,500 × (1 - 0.005) = 43,065 USDT
→ 点差收益：870 USDT
```

## 🆘 常见问题

### Q1: 如何在两个功能间切换？

**A**: 
点击顶部的 Tab 按钮:
- "汇率管理" → 查看和管理交易对
- "费率模板" → 查看和管理模板

---

### Q2: 为什么我的搜索没有结果？

**A**: 
检查以下几点:
1. 确认在当前正确的 Tab
2. 检查筛选状态（是否只显示已启用）
3. 尝试更短的关键词
4. 清除搜索框重新输入

---

### Q3: 应用模板后可以手动修改吗？

**A**: 
✅ 可以！模板只是快速填充工具
- 应用后仍可修改任何字段
- 修改后保持独立
- 不受模板变更影响

---

### Q4: 如何查看模板被哪些交易对使用？

**A**: 
- 当前版本：显示使用数量
- 详细列表：后续版本支持
- 临时方案：记录模板使用情况

---

### Q5: 可以同时打开两个 Tab 吗？

**A**: 
- ❌ 同一时间只能看到一个 Tab
- ✅ 但可以快速切换
- ✅ 切换时状态会保持
