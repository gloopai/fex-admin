# 费率模板快速上手指南

## 🎯 5 分钟快速开始

### 第一步：创建第一个费率模板

1. **访问路径**: 资金管理 → 费率模板
2. 点击 **「+ 新增模板」**
3. 填写基本信息:
   ```
   模板名称：我的标准模板
   模板类型：标准费率
   描述：适用于大部分交易对
   启用状态：✓
   ```
4. 展开 **「基础手续费与价差」**:
   ```
   买入手续费率：0.5%
   卖出手续费率：0.5%
   ```
5. 展开 **「分级费率设置」** (可选):
   - 系统已自动填充 5 级费率
   - 可根据需要微调
6. 点击 **「保存」** → 通过 MFA 验证

✅ **完成!** 你创建了第一个费率模板

---

### 第二步：在汇率管理中应用模板

1. **访问路径**: 资金管理 → 汇率管理
2. 点击 **「+ 新增交易对」** 或编辑现有交易对
3. 在「基本信息」区域，找到 **「💰 快速应用费率模板」**
4. 选择刚才创建的 **「我的标准模板」**
5. 系统自动填充:
   - ✅ 买入费率：0.5%
   - ✅ 卖出费率：0.5%
   - ✅ 5 级分级费率
6. 继续填写其他信息（资产对、数据源等）
7. 点击 **「保存」** → MFA 验证

✅ **完成!** 成功应用费率模板

## ❗ 注意事项

### ⚠️ 重要提醒

1. **模板修改不影响已应用的交易对**
   - ❌ 修改模板 ≠ 自动更新交易对
   - ✅ 需要手动重新应用模板

2. **MFA 验证是必须的**
   - 所有模板操作都需要 MFA
   - 确保手机/验证器可用

3. **分级费率的独立性**
   - 可以手动调整各级别费率
   - 调整后不再跟随模板变化

4. **模板删除需谨慎**
   - 建议禁用而非删除
   - 保留历史记录便于追溯

## 🆘 常见问题速查

### Q: 如何快速创建多个相似模板？

**A**: 
1. 创建第一个模板
2. 点击「编辑」→ 复制内容
3. 修改费率和名称
4. 保存为新模板

---

### Q: 应用模板后可以手动调整吗？

**A**: 
✅ 可以！模板只是快速填充工具
- 应用后仍可手动修改任何字段
- 修改后保持独立性
- 不受模板变更影响
